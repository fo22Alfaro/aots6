## ¿Qué es AOTS⁶?

Framework original para representación ontológica distribuida en manifold toroidal T⁶ = (ℝ/ℤ)⁶ con identidad criptográfica SHA-256 y protocolo peer-to-peer sin autoridad central.
T⁶ = D0(tiempo) × D1(espacio) × D2(lógica) × D3(memoria) × D4(red) × D5(inferencia)
